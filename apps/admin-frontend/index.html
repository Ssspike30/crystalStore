crystalStore / admin frontend

把订单、发货、退款三条运营动作，压进一个能直接联调的静态管理台。

这不是后台空壳页，而是围绕现有 admin-backend 接口做的最小可用运营台。 你可以直接连接真实接口，也可以在接口不可用时切到本地 demo 模式演示整条流程。
